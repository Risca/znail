Network Emulator

The Network Emulator can be used to emulate various network conditions and can manipulate the flow of packets on the network.

graph LR network --- eth0 subgraph network-emulator eth0 --- br0 br0 --- eth1 end eth1 --- sut

To be able to affect network behaviour, the Network Emulator is connected to the SUT and the network that the SUT should access is connected to the Network Emulator. In this configuration, the Network Emulator effectively acts as a network switch.

To manipulate network packets, the Network Emulator makes use of a collection of standard Linux tools:

  • iptables is used to direct traffic flow
    • iptables
  • tc, netem and tc-tbf are used to emulate network conditions.
    • tc
    • netem
    • tc-tbf
